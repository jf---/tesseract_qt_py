# tesseract_qt_py

Python Qt6/VTK replacement for tesseract_qt. ~7.3k LOC vs 59k C++ LOC.

## Setup

```bash
pixi install
pixi run check  # verify deps
pixi run app    # launch viewer
pixi run test   # run tests
```

Requires pre-built `tesseract_nb` conda environment with tesseract-robotics-nanobind.
